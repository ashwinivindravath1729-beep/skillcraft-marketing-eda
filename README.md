# 📊 Marketing Campaign - Business Insights Report (EDA)

## 📌 Task 04 - SkillCraft Technology Internship

## 🎯 Objective
Perform Exploratory Data Analysis on a Marketing Campaign Dataset and recommend which channels should receive more budget based on ROI.

## 📂 Dataset
- **Source:** Marketing Campaign Dataset (Generated)
- **Rows:** 1,000 records
- **Columns:** 12 (customer_id, age, gender, channel, campaign, impressions, clicks, conversions, spend, revenue, region, ROI)

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## 📊 Analysis Performed
- ✅ Data Loading and Basic Info
- ✅ Missing Value Check
- ✅ ROI by Marketing Channel
- ✅ Revenue by Channel
- ✅ Conversions by Campaign (Pie Chart)
- ✅ Spend vs Revenue Analysis
- ✅ Marketing Funnel Visualization
- ✅ 1-Page Business Insights Report

## 🔍 Key Insights
- 📌 **Paid Ads** has highest ROI (294.03%)
- 📌 **Email** generates maximum revenue ($1,46,678)
- 📌 **Social Media** has lowest ROI (241.74%)
- 📌 Overall Average ROI is 270.20%

## ✅ Recommendations
1. Increase budget for **Paid Ads** — highest ROI!
2. **Email** generates maximum revenue — scale up!
3. Reduce spend on **Social Media** — lowest ROI performer
4. Focus on conversion optimization across all channels
